# DSA Practice Log

This repository is a structured log of my daily data structures and algorithms practice. I use it to stay consistent, improve problem-solving speed, and prepare for technical interviews.

## Purpose

The goal of this repo is not just to store accepted solutions, but to track how I approach problems, improve patterns, and write cleaner code over time.

I am focusing on common interview topics such as:

* Arrays and Hashing
* Two Pointers
* Sliding Window
* Stack
* Binary Search
* Linked Lists
* Trees
* Graphs
* Dynamic Programming
* Backtracking
* Greedy Algorithms
